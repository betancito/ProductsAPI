Welcome to the Products API

This API allows you to manage and retrieve product data in an easy and efficient manner.

Below are the available API endpoints that you can use to interact with the API:

API Endpoints

Endpoint Method Description
/api/v1/products GET Retrieves a list of all products.
/api/v1/products/{id} GET Retrieves a specific product by its ID.
/api/v1/products POST Creates a new product.
/api/v1/products/{id} PUT Updates a product by its ID.
/api/v1/products/{id} DELETE Deletes a product by its ID.
